    Dasar-Dasar Zig#

    Sebelum mulai menulis program kompleks, penting untuk memahami bahwa Zig sangat menghargai kejelasan dan eksplisit. Berikut adalah langkah awal untuk memulai pembelajaran Zig:

    1. Struktur Program#

    Setiap program Zig yang dapat dieksekusi (executable) dimulai dengan fungsi main. Berbeda dengan C, Zig menggunakan pustaka standar (std) yang diimpor secara eksplisit menggunakan bawaan @import.

    const std = @import("std");

pub fn main() !void {
    // getStdOut() digunakan untuk mencetak langsung ke standar output.
    const stdout = std.io.getStdOut().writer();
    
    // try digunakan karena operasi penulisan (print) bisa saja gagal (menghasilkan error).
    try stdout.print("Halo, dunia Zig!\n", .{});
}
